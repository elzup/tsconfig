# tsconfig

> Shared [TypeScript config](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) for my projects

## Install

```
$ npm install --save-dev @elzup/tsconfig
```

## Usage

`tsconfig.json`

```json
{
	"extends": "@elzup/tsconfig/tsconfig",
	"compilerOptions": {
		"outDir": "dist"
	}
}
```
